--------------------------------------
🚀 CÓMO SUBIR EL SITIO A UN HOSTING
--------------------------------------

1. ✅ Reuní todos los archivos (incluidos los subdirectorios css/, js/, images/)
2. 📦 Comprimilos en un ZIP o cargalos directamente desde el administrador de archivos de tu hosting
3. 🔼 Usá alguno de estos métodos para subirlo:
   - Acceso FTP (como FileZilla)
   - Panel web (como cPanel, Hostinger, GoDaddy)
   - GitHub Pages (para hosting gratuito y rápido)

4. Verificá que tu archivo principal sea:
   → `index.html`

--------------------------------------
🌐 CÓMO COMPRAR UN DOMINIO
--------------------------------------

Un dominio es el nombre personalizado de tu web (ej: www.elegea.com.ar)

1. 💳 Sitios recomendados para comprar dominios:
   - https://www.donweb.com
   - https://www.hostinger.com.ar
   - https://nic.ar (para dominios .com.ar)

2. 👣 Pasos básicos:
   - Elegí el nombre (que esté disponible)
   - Comprá o registrá (puede ser anual)
   - Apuntalo a tu hosting o conectalo a tu cuenta de Google Sites / GitHub / cPanel

3. 💡 Tip:
   - Si usás .com.ar, debés validarlo desde AFIP con CUIT o clave fiscal.
   - Para uso global, elegí .com / .net / .online
